# 🏨 SmartStay HMS

A desktop-based Hotel Management System developed using **Python**, **Tkinter**, and **MySQL**.

---

## 📌 Project Overview

SmartStay HMS is designed to simplify hotel management by providing modules for customer management, room booking, hall booking, billing, and record management through a user-friendly desktop interface.

---

## ✨ Features

- 🔐 Secure Login
- 👤 Customer Management
- 🛏️ Room Booking
- 🏛️ Hall Booking
- 💰 Billing System
- 🔍 Search Records
- ✏️ Update & Delete Records
- 🗄️ MySQL Database Integration

---

## 🛠️ Technologies Used

- Python
- Tkinter
- MySQL
- Pillow

---

## 🚀 How to Run

1. Clone the repository

```
git clone https://github.com/Sachichhoker/SmartStay-HMS.git
```

2. Install required libraries.

3. Import the MySQL database.

4. Run

```
login.py
```
